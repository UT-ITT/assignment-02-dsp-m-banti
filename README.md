# READ ME
This Repository was written on a Macbook apple silicon chip (M4).

# Start Guide
1. Clone the repository
2. Create a Virtual environments from the "requirements.txt" file
3. Have fun trying

## Karaoke Game Guide
Start the game with:
```python3 karaoke_game/karaoke.py```

### Disclaimer
I understood teh assignment wrong and did first a game based on pitch without a "song". It is nearly finished in the karaoke_rocket.py file. I didn't finish it to start a new Karaoke.py file with the right excercise description.

## Whistle Input Guide
Start the control with:
```python3 whistle_input/whistle-input.py```
